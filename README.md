# 🪐 AURA OS v.99

A chaotic, fully interactive Y2K / Vaporwave desktop environment built entirely in one file with HTML, CSS (Tailwind), and pure JavaScript.

## 🚀 Features
* **Draggable Window System:** Open, close, and stack apps just like a real retro OS.
* **Flappy Alien Mini-Game:** A physics-based canvas game built from scratch.
* **Cursed Paint:** A functional neon drawing canvas with glowing brushes.
* **Tamagotchi Pet:** Keep the pet fed with Shawarmas so its aura doesn't hit zero.
* **Brainrot Oracle:** A randomized quote generator for instant enlightenment.

## 🛠️ Tech Stack
* HTML5 / Canvas
* Tailwind CSS
* JavaScript (ES6+)
